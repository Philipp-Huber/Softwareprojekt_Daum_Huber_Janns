Detailed Description

The PeptideView class.

The PeptideView class offers a customized version of the QTableView class. In addition to the QTableView functions it offers a slot which can hide and show rows depending on given entries.

◆ toBeDisplayed

void PeptideView::toBeDisplayed ( QList< QString >  displayThese)
slot

This function hides and shows rows according to its input.

toBeDisplayed receives a List of accesion codes as QStrings. If the model contains a column with header "accession" the function will set all rows as shown, for which the entries in the accession column are contained in the input, and set all columns as hidden for which the entries aren't. If the model does not contain such a column, the function will return without doing anything. This function can be connected as a Slot
